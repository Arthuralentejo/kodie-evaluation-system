## CRMs open source (auto‑hospedados)

### Odoo CRM
- **Integração e dashboards:** módulo de CRM com pipeline visual, automação e relatórios em tempo real; integra via API com outros sistemas e com demais apps (ERP, site, e‑commerce etc.).
- **Complexidade:** média/alta; demanda servidor, conhecimento técnico (DevOps + Python) e tempo de customização, mas dá bastante flexibilidade. 

### SuiteCRM / SugarCRM (comunidade)
- **Integração e dashboards:** CRMs web muito personalizáveis, com módulos, campos customizados, workflows e APIs REST para integração com sistemas externos.
- **Complexidade:** alta; poderosos para cenários complexos, porém exigem equipe técnica para instalação, atualizações e customizações mais profundas. 

### Dolibarr (ERP + CRM)
- **Integração e dashboards:** suite web ERP/CRM para empresas e fundações, com módulos de contatos, faturas, projetos e e‑mailings; possui APIs e pode expor dados para outras ferramentas de BI.
- **Complexidade:** média; instalação relativamente simples em servidor PHP, mas customizações avançadas pedem desenvolvedor com experiência na stack do projeto.

## CRMs em nuvem para ONGs (pago, com foco terceiro setor)

### Zoho CRM para ONGs
- **Integração e dashboards:** API REST completa, marketplace com integrações, automações, relatórios e dashboards personalizáveis; tem oferta e documentação específica para organizações sem fins lucrativos.
- **Complexidade:** baixa/média; configuração via interface, com construção de telas, campos e dashboards sem código, mas integrações mais avançadas podem requerer desenvolvedor (functions, webhooks).

### CRMs específicos para ONG (ex.: HYB CRM, Trackmob)
- **Integração e dashboards:** focados em cadastro de doadores, régua de relacionamento e relatórios de repasses, com funções de comunicação e funis para terceiro setor.
- **Complexidade:** baixa; normalmente já vêm com fluxos prontos para ONG, mas costumam ter menos flexibilidade que um CRM genérico de grande porte. 

## CRMs SaaS generalistas com forte integração e dashboards

### HubSpot CRM
- **Integração e dashboards:** possui CRM gratuito com API robusta, marketplace de apps e possibilidade de dashboards customizados (relatórios e painéis personalizados para vendas, marketing e atendimento).
- **Complexidade:** baixa; a configuração básica de pipelines e dashboards é simples via interface, mas integrações customizadas (via API, webhooks, “custom cards”) pedem desenvolvedor.

### Jestor (no‑code + dashboards)
- **Integração e dashboards:** plataforma no‑code para criar “mini‑sistemas internos” com dashboards inteligentes, integrando dados de CRM, finanças, marketing etc., com foco em automação e relatórios.
- **Complexidade:** baixa/média; construção dos fluxos é visual, com pouca necessidade de código, mas projetos maiores exigem boa modelagem de dados.
